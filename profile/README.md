## üìö Quick Start Guide

1. Download the latest version of **Voicemeeter Potato** from the official website.
2. Install the application, which includes the necessary virtual audio driver.
3. Restart your computer as recommended to ensure driver stability.
4. Launch **Voicemeeter Potato** and configure your hardware inputs (microphones, interfaces) in the top "Hardware Input" sections.
5. Set your physical outputs (speakers, headphones) in the bottom "Hardware Out" sections (A1, A2, etc.).
6. Set your default Windows playback device to "Voicemeeter Input (VAIO)" and your recording device to "Voicemeeter Output (VAIO)" for system-wide control.
7. Route audio by using the matrix buttons below each strip, sending desired sources to your chosen outputs.
8. Use the macro buttons to save and recall different audio profiles for gaming, streaming, or music listening.
